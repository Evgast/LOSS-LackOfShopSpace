# LOSS-LackOfShopSpace, because LOS or LOSSB wouldn't shound good
A Balatro mod that provides more space for buttons in shop
Utilizes SMODS.UIScrollBox to let you scroll buttons with a mouse wheel.
Also technically provides an API for adding those buttons, simply add your button UI code to one of the three tables provided in main.lua. 
Expect the mod to be compatible with mods that patch in their buttons without patching out the whole damn shop lol.
